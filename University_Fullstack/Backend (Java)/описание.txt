Магазин:

Аутентификация (Вход через пару  логин/почту и пароль. Логин и почту можно вводить в одно поле)

Личный кабинет для простого пользователя (список товаров, поиск по артикулу, Корзина (добавление и удаление товаров из корзины, просмотр))
Личный кабинет для администратора (список товаров, добавление удаление товаров из базы, список пользователей, изменение прав)

регистрация (проверка логина в базе, проверка почты в базе, ввод почты по маске)
