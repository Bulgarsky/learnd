/widget.bank.calc.depo-interest

Калькулятор депозита













Описание:

Формула простого процента (без капитализации вклада): доход = (депозит * ставка * срок размещения(в днях) / 365 ) / 100;
S=(P*I*t/K)/100
K — число дней в году (365)

Формула сложного процента (с капитализацией вклада): вклад = ((вклад(и проценты) * ставка * срок капитализации) / 365 ) / 100;
S=(P*I*j/K)/100
K — число дней в году (365)

Значения:
месяц = 30; квартал = 120; год = 365;

Ограничения полей:
Сумма вклада: мин = 10 000; макс = 100 000 000; шаг = 100 000;
Срок: мин = 1; макс = 3660; шаг = 30;
Стака: мин = 0,25%; макс = 30%; шаг = 0,25%;,
