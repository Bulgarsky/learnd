Descendant combinator. Селекторы потомков. потомок Только 1-го уровня!

The " " (space) combinator selects nodes that are descendants of the first element.
Syntax: A B
Example: div span
will match all span elements that are inside a Div element.

Child combinator. Дочерние Селекторы

The > combinator selects nodes that are direct children of the first element.
Syntax: A > B
Example: ul > li
will match all li elements that are nested directly inside a ul element.

    ul tag
  • li 1
  • li 2
  • li 3
