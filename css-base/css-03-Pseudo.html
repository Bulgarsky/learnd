Pseudo-class & pseudo-element

Псевдо классы

  • root {атрибут:значение} - позволяет выбрать корневой элемент веб-страницы(почти всегда html)
  • focus {атрибут:значение} - фокус при табуляции, нажатии мыши на поле ввода
  • empty {} выбирает gecnst элементы, которые не имеют вложенных элементов
  • not - позволяет исключить элементы из списка элементов, к которым применяется стиль
  • lang - стилизует элементы на основании значения атрибута lang
    A: псевдоклассы для ссылок
  • link {атрибуты} - обычное состояние, без перехода
  • visited {атрибуты} - состояние "посещен"
  • active {атрибуты} - состояние непосредственно при переходе
  • hover {атрибуты} - состояние при наведении указателя мыши
